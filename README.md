# agentic-plugins

Personal Claude Code plugins, config, and scripts.

## Setup

```bash
make sync   # symlink config + plugins into ~/.claude/
```

## What's here

### Config

- **statusline-command.sh** — Status line showing model name, context window usage (color-coded), and session cost

### Plugins

- **compile-sessions** — Extract Claude Code session transcripts into readable markdown (WIP)
